1. Laden sie sich das Processing Framework von folgendem Link herunter und installieren sie dieses: https://processing.org/de/download/
2. Starten sie die "sim2dProcessing.exe" in dem Ordner /sim2dProcessing/application.windows64/.
3. Installieren sie die aktuelle Java SDK auf Ihrem System: https://www.oracle.com/java/technologies/downloads/
4. Öffnen sie die Lighthouse_Controllerv2 Ordner mit einem beliebigen Java Editor und starten sie dann die main
5. Schließen sie den Leapmotion Controller an ihren PC an und installieren sie die zugehörige Software: https://developer.leapmotion.com/tracking-software-download
6. Starten sie die .exe des Unity Programms um den Leapmotion Controller mit unserem Framework zu verknüpfen.
7. Folgende Gesten führen zu folgendem Output am LightHouse:
 - nach links oder rechts wischen -> Welle
 - "Mamma Mia"(https://ergo-reiseblog.de/laendertipps/italienische-gesten) mit linker Hand -> Equalizer-Balken darstellen
 - "Mamma Mia"(https://ergo-reiseblog.de/laendertipps/italienische-gesten) mit rechter Hand -> Superman Animation

8. Have Fun!
